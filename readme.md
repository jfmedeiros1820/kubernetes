## Start Minikube
`minikube start`

## Stop Minikube
`minikube stop`

## Create new object
`kubectl create -f <file>`

## List all pods
`kubectl get pods`

## Remove pode
`kubectl delete pods <Pod Name>`

## Visualize Pods info
`kubectl describe pods <Pod Name>`

## Open Dashboard
`minikube dashboard`

## Visualize service url
`minikube service servico-aplicacao --url`
